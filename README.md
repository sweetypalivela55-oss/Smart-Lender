 🏦 Smart Lender

A Machine Learning based Loan Approval Prediction Web Application built using **Python, Flask, HTML, CSS, and Scikit-learn**.

📌 Project Overview

Smart Lender predicts whether a loan application is likely to be approved based on the applicant's information. It helps automate loan approval decisions using a trained Random Forest Machine Learning model.

 🚀 Features

- Loan Approval Prediction
- Machine Learning Model (Random Forest)
- Flask Web Application
- User-Friendly Interface
- Fast and Accurate Predictions
- Responsive Web Pages

 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- Scikit-learn
- NumPy
- Pickle

 ⚙️ Installation

Clone the repository:


git clone https://github.com/sweetypalivela55-oss/Smart-Lender.git


Go to the project folder:


cd Smart-Lender


Install dependencies:


pip install -r requirements.txt


Run the application:


python app.py


Open your browser:


http://127.0.0.1:5000


 📈 Machine Learning Model

- Algorithm: Random Forest Classifier
